# **App Name**: AgriAssist

## Core Features:

- Crop Recommendation: Suggest suitable crops based on farmer's location and soil type input.
- Fertilizer Recommendation: Provide fertilizer recommendations based on the suggested crops.
- Image-based Disease Detection: Allow farmers to upload crop images for instant disease detection using a machine learning tool.
- Weather-based Alerts: Provide real-time alerts for weather changes affecting crops, via integration with weather APIs.
- User Input Interface: A simple form-based interface for farmers to input location and soil type.
- Advisory Display: Display crop and fertilizer recommendations in an easy-to-read format.
- Alert Notifications: Push notifications to inform farmers about weather hazards, pest attacks, or disease outbreaks.

## Style Guidelines:

- Primary color: A muted earthy green (#8FBC8F) to reflect nature and agriculture.
- Background color: Light beige (#F5F5DC) to provide a soft and natural backdrop.
- Accent color: Warm gold (#D4A27A) for highlighting important information and calls to action, creating contrast.
- Headline font: 'Playfair', a serif font lending an elegant, high-end feel; body font: 'PT Sans' sans-serif
- Use clean, line-based icons related to weather, crops, and soil conditions.
- Clean and simple layout, optimized for mobile devices with clear sections for input, recommendations, and alerts.
- Subtle transitions and animations to enhance user experience without being distracting.
